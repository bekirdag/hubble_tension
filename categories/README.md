## Reading Order

1. Start with `08_reviews_comparisons_no_go.md` to understand the global map and why some fixes are disfavored.
2. Read `01_early_universe_pre_recombination.md` for the main new-physics route that changes the CMB-inferred sound horizon.
3. Compare it with `02_late_universe_dark_energy_modified_gravity.md` and `06_inverse_distance_ladder_bao_expansion_history.md`, because late-time fixes are tightly constrained by BAO and supernova distances.
4. Use `03_distance_ladder_calibration_systematics.md` and `04_independent_late_universe_h0_probes.md` to evaluate whether the local value is biased or independently confirmed.
5. Use `07_growth_s8_lensing_clusters_cmb_lensing.md` as the compatibility check: viable H0 solutions should not worsen S8, lensing, or cluster-count tensions.
