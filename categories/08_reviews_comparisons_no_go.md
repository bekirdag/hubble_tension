# Reviews, Model Comparisons, and No-Go Results

## Solution Idea

This category collects papers whose main method is comparative rather than proposing one mechanism:
reviews, buyer guides, model rankings, no-go theorems, geometric-obstruction arguments, and broad
critiques. They are useful as navigation papers because they explain why some categories are
promising and why others fail once all data are combined.

## Main Tradeoff

These papers should be read as maps of the solution space. They usually depend on the data set
choices and model priors available at the time they were written.
